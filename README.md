# Kaggle_Study-
잇츠 IT's 스터디 3기 '데이터 공모전 메달리스트 도전기' 

## ✨ 스터디 소개

**잇츠스터디 3기**는 서울시여성가족재단이 주관하는 학습형 스터디 프로그램입니다.  
본 팀 **‘데이터 공모전 메달리스트 도전기’**는 실전 데이터 분석 공모전에서의 메달 획득을 목표로, 실무 기반의 프로젝트 수행을 통해 데이터 분석 역량을 체계적으로 향상시키고자 합니다.

스터디는 매 회차마다 참가자가 순차적으로 참고 도서의 챕터를 발표하고, 이를 바탕으로 Kaggle 대회에 참여하는 방식으로 운영됩니다.  
이러한 과정을 통해 실전 감각을 익히고, 팀원 간 협업 및 문제 해결 능력을 강화하는 것을 지향합니다.
